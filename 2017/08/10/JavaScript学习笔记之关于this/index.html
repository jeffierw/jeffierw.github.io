JavaScript学习笔记之关于this

发表于 2017-08-10 | 分类于 前端 ， js | 阅读次数:

前言

在《JavaScript学习笔记之执行上下文栈》中讲到，当JavaScript代码执行一段可执行代码(executable code)时，会创建对应的执行上下文(execution context)。

对于每个执行上下文，都有三个重要属性

  • 变量对象(Variable object，VO)
  • 作用域链(Scope chain)
  • this

今天讲讲 this

this指向

我们可以简单的理解 this 为调用函数的对象，

1、作为对象调用时，指向该对象

2、作为函数调用, 指向全局window

3、作为构造函数调用,指向当前实例对象

4、作为call与apply调用,

目前水平有限，对于this的理解只能从调用场景上来理解，未来理解深刻时再补充。
